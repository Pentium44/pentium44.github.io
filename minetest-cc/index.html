About
Minetest-CC is a sandbox game based from Perttu "celeron55" Ahola's Minetest-c55 pre-releases. So far, the game has been modified with new nodes (Blocks), tools, and features. Some new features include:
  • New clean textures
  • New randomized mineral generation
  • New minerals (Gold)
  • New crafting recipes
The minimum requirements for Minetest-CC hardware are:
  • 512MB Ram
  • Celeron / Pentium4™ 1.8 Ghz CPU
  • 64MB VRam (Updated drivers / OpenGL support)
  • Lan Connection (For Multiplayer)
