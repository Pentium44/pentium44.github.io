SpeeDL - Speed Downloader

SpeeDL is a simple file transfer server based on the HTTP protocol. The client is based on the libcURL libraries while the server uses basic C sockets and functions. SpeeDL servers support file list pushing and file pushing when requested by a SpeeDL client. The SpeeDL client supports file grabbing from the server, list retrieval from the server, and has a refreshing download meter.

######### SpeeDL #########

A simple file sharing server and client powered by cURL. SpeeDL
runs natively on GNU/Linux distros and can also be cross compiled
on windows machines. The default SpeeDL port is 2750

Compiling:
	1- Download.
	2- run: "make" command
	3- start server.

Starting a server:
	Compile and run:
		speedl-server [port] [server directory]
		EX: speedl-server 2750 files/

Downloading from a server: 
	Compile and run:
		speedl 
 [port] 

		Ex: speedl www.example.com archive.zip
			Or if on a non-standard port
			speedl www.example.com 2751 archive.zip
		The download will be available in the working directory.
		
		Getting server's file list:
			speedl www.example.com list
