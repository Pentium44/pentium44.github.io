Chris's Projects - CTMB

CTMB is a simple, feature-filled, and secure forum software that has been worked on for the better of the last two years. It has XSS security fixes, hashed passwords, flatfile database, user signatures, and more. Scroll down for a full list of features, downloads, information, changelog and more.

Features

Here is a full list of features that are included in CTMB currently:
- Flatfile database for speed (Secure)
- Hashed passwords (sha1 and md5)
- Properly stored user information (Cannot be accessed by other users)
- Administration
- Topics can be deleted
- Categories
- User ranks
- User colors
- User signatures
- Avatar support
- User list
- Theme support
- User based theme choosing
- User friendly installation
- Lightweight
- Runs on Linux, Windows, and Mac (PHP 5.1+ recommended)
- BBCode support
