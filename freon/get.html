Freon is built on top of libc and busybox to create a useful, powerful OS that is also light and nimble. Freon Linux is, and will remain, free and open source software. All of the scripting, and software developed for Freon including config scripts and the OS' build script can be located on Freon Linux's Github repository. Freon's base source (glibc, busybox, linux, etc.) can be found here. Need package sources? No problem, find them here.
