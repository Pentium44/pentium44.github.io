Chris's Projects

This is a website to present my software and scripts that could be helpful to you. Project summaries:
CTMB CTMB, also known as Crazy Tiny Message Board, is a simple yet fully functional discussion board software open to the world and is released under the CC-BY-NC 3.0.
Chttpd Chris's HTTP Daemon is a simple web server written in C for *NIX operating systems. Chttpd supports basic CGI and static web page documents. Chttpd is released under the GPLv2+.
PentDB PentDB is a simple database engine that is still under major development. It requires PHP and only PHP (5.2+ recommended). This software is released under the GPLv3.
